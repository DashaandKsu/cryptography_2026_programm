1. Логика шифрования Тритемия

Формула: Yj = Xi+j-1 mod n
X – исходный текст
Y – зашифрованный текст
i – номер буквы в алфавите
j – номер буквы в тексте
n – длина алфавита

Проще говоря, каждая буква исходного текста заменяется буквой из таблицы Тритемия: для j-й буквы текста берём строку алфавита, сдвинутую на позицию j-1.

То есть нужно:
Подготовить текст (с помощью функций из Атбаш)
Преобразовать каждую букву в индекс в алфавите
Использовать таблицу Тритемия для выбора зашифрованной буквы
Собрать зашифрованный текст
При желании разделить на группы по 5 символов (как в Атбаш)

2. Какие функции нам нужны
prepare_text_for_encryption(text, punctuation_codes) – из Атбаш, готовит текст
split_into_groups_of_five(text) – из Атбаш, разбивает текст для удобного отображения
create_punctuation_codes() – из Атбаш, для подготовки текста

Новые функции для Тритемия:
encrypt_tritemius(text, alphabet) – шифрует текст по таблице Тритемия
decrypt_tritemius(text, alphabet) – расшифровывает текст (обратный процесс)

# Проект: лабораторные по криптографии

## Контекст
- Студентка 2 курса ИБ, дисциплина «Программирование криптографических алгоритмов»
- Лабораторные по шифрам: Атбаш, Цезарь, Полибий, Тритемий, Виженер, Магма и т.д.
- Каждая лаба → код шифра на Python + блок-схема по ГОСТ 19.701-90 + отчёт в Word/PDF

## Окружение
- ОС: Windows 11
- Терминал: PowerShell (не bash)
- Редактор: VS Code
- Python: основной язык
- Базы данных: PostgreSQL, MySQL (для других дисциплин, не криптографии)
- Виртуальные машины (для сетей, не криптографии): Kali Linux, RedOS, Ubuntu Server

## Работа с блок-схемами
**При любой задаче по созданию блок-схемы обязательно читать требования:**
@docs/blockscheme_requirements.md

Краткая выжимка (полные правила в файле выше):
- Формат вывода — только `.drawio`
- Циклы — только шестиугольниками `loop_limit`, никогда ромбами
- Все блоки белые с чёрным контуром
- Имена функций и переменных как в коде, описания внутри блоков на русском
- Шифрование и расшифрование на одной схеме
- Каждая функция из кода должна быть в схеме хотя бы блоком-палочкой
- Готовые `.drawio` складывать в `flowcharts/`

## Работа с кодом
- Стиль кода — простой и читаемый, как пишет студент 2 курса
- Не использовать продвинутые конструкции Python без необходимости
- Комментарии на русском
- Если в коде есть валидация ввода — обязательно отражать её в блок-схеме

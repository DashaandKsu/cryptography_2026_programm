Есть задание для создания программы, которая будет представлять все заданные шифры, которые будут находиться в отдельных папках, таких как lab_1, labb_2 и так далее.
В этой программе нужно выбрать один из представленных шифров, который программно зашифрует введенный с клавиатуры литературный текст (важно, чтобы в нем не было буквы "ё" либо же она заменена на букву "е") на тысячу символов и расшифрует также написанный с клавиатуры шифртекст.
К текстам есть еще требования: 
1. сначала происходит ввод нашей фразы полностью, с пробелами, точками, запятыми, тире и так далее.
2. программа должна убрать все пробелы между буквами, а все символы заменить на короткое значение из трех символов, например: знак точки - "тчк", знак запятой - "зпт" и так далее. Если в тексте будут другие символы это лучше уточнять как их правильно переделывать
3. после шифрования каждой буквы полученного текста нужно текст разбить на группы по пять символов и между ними пробел (представление в консоли для удобства)
4. сразу же должно быть расшифование нашего только что зашифрованного текста так, чтобы слова были полноценными, не склеивались и все знаки препинания такие как "тчк" и "зпт" должны вернуться в первоначальный вид символа, такие как "." или ","
5. для тестирования нужно будет использовать текст объемом 1000 символов для зашифроки, а также расшифровки.
В программе обязательно должны быть комментарии, не сильно большие, но очень понятные для понимания.
Если шифр использует ключ, то обязательно должна быть проверка на то, что заданный ключ введен верно.
По программе нужно сделать блок-схему ее работы.
Язык программирования - Python.
По итогу работы со всеми шифрами должно получиться диалоговое окно с выбором варианта шифра для зашифровки любого литературного текста.
Формат параметро - 32 буквы русского алфавита, без буквы "ё", также можно использовать таблицу аски кода, либо использовать расширенный алфавит для шифровки символов, используя 64 знака.
При расшифровании важно читаемость расшифрованного текста.
Найти текст на тысячу знаков, который будет использоваться для теста всех программ.

Тестирование программы: быть уверенным, что работает именно тот алгоритм, который был заказан, а также проверять уже зашифрованный и расшифрованный раннее текст, который мы шифровали без программы, с помощью математики.

Писать код надо как студент, просто и понятно. знакомый только с базовыми конструкциями языка, избегая сложного синтаксиса. Максимально разделяя код на более простые блоки. Для каждого блока написать короткий комментарий с объяснением в контексте всей задачи шифрования.

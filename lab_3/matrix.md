Мне нужно реализовать матричный шифр (шифр Хилла) на Python для лабораторной работы. Код должен быть максимально простым и понятным для студента. Без numpy и без сложных конструкций.

У меня уже есть проект с меню main.py, где каждый шифр лежит в отдельном файле и содержит две функции:

encrypt_... (text)

decrypt_... (text)

Также у меня есть общий файл с функциями обработки текста, которые используются во всех шифрах. Я ОБЯЗАТЕЛЬНО хочу использовать их и не дублировать код.

Вот функции, которые уже существуют и которые нужно использовать:

create_punctuation_codes()

create_code_to_punctuation()

prepare_text_for_encryption(text, punctuation_codes)

split_into_groups_of_five(text)

restore_punctuation_from_codes(text, code_to_punctuation)

Логика работы такая же, как в других моих шифрах (например, Тритемий):

ШИФРОВАНИЕ:

Получаем словарь знаков препинания.

Подготавливаем текст через prepare_text_for_encryption.

Работаем только с буквами русского алфавита:
RUS_ALPHABET = "абвгдежзийклмнопрстуфхцчшщъыьэюя"

Буквы заменяем на их индексы (0–31).

Разбиваем текст на блоки по 3 символа (матрица 3×3).

Если длина не кратна 3 — дополняем последней буквой алфавита.

Каждый блок преобразуем в числовой вектор.

Умножаем вектор на матрицу-ключ:

A = [
[1, 4, 8],
[3, 7, 2],
[6, 9, 5]
]

Результат берём по модулю 32.

Полученные числа переводим обратно в буквы.

Разбиваем результат на группы по 5 символов.

Возвращаем строку в верхнем регистре.

РАСШИФРОВАНИЕ:

Убираем пробелы между группами.

Переводим в нижний регистр.

Используем обратную матрицу A^-1 (по модулю 32).

Разбиваем текст на блоки по 3.

Умножаем каждый блок на обратную матрицу.

Берём по модулю 32.

Переводим числа обратно в буквы.

Восстанавливаем знаки препинания через restore_punctuation_from_codes.

Возвращаем читаемый текст.

Важно:

При расшифровке мы не "знаем" исходный текст.

Восстановление пробелов и знаков должно происходить только через коды.

Не использовать numpy.

Написать свои функции:

умножение матрицы на вектор

вычисление определителя 3×3

нахождение обратного элемента по модулю

нахождение обратной матрицы 3×3 по модулю 32

Структура файла должна быть:

lab_3/matrix_cipher.py

Внутри должны быть:

encrypt_matrix(text)

decrypt_matrix(text)
